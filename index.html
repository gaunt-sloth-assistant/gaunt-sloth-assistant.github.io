Gaunt Sloth AI Assistant

Gaunt Sloth Assistant is a lightweight command line AI assistant for software developers, designed to enhance code review quality while reducing cognitive load and time investment on code reviews and pull request diff analysis.
View on GitHub | View on NPM

Documentation

  • Reviews code and suggests bug fixes
  • Explains provided code
  • Reviews diffs and pull requests
  • Fetches requirements from GitHub issues or Jira
  • Answers questions about code
  • Writes code and saves responses in markdown
  • Provider flexibility: choose between Vertex AI, Anthropic, Groq
  • Open-source transparency, no vendor lock-in
  • Command-line integration and extensibility
  • Specialized for code review and PR analysis
  • Supports Model Context Protocol (MCP)
  • Cost effective and efficient
